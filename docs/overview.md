# GoRogue - プロジェクト概要

## 概要

GoRogueは、Go言語とGruidライブラリを使用して開発される本格的なローグライクゲームです。伝統的なローグライクゲームの要素を忠実に再現し、手続き生成ダンジョン、ターンベース戦闘、パーマデス（永続的死）、探索重視のゲームプレイを提供することを目的としています。

## プロジェクトの目的

- **伝統的なローグライクゲーム体験の提供**: オリジナルRogueの26階層構造を再現
- **Go言語によるゲーム開発の実践**: Goの特徴を活かしたシンプルで効率的な実装
- **Gruidライブラリの効果的な活用**: ローグライクゲームに特化したライブラリの活用
- **段階的な開発・学習の実現**: 基本機能から高度な機能まで順次実装

## 主要な特徴

### 計画されているゲーム機能
- **26階層の手続き生成ダンジョン**: オリジナルRogueに準拠した階層構造
- **ターンベース戦闘システム**: 戦略性を重視した戦闘メカニクス
- **包括的なアイテムシステム**: 武器、防具、ポーション、巻物、食料、指輪
- **モンスターシステム**: 多様なモンスターとAI
- **状態異常システム**: 毒、麻痺、混乱などの実装
- **トラップシステム**: 落とし穴、毒針、テレポートトラップ
- **セーブ・ロード機能**: ゲーム状態の永続化
- **パーマデス**: 真の一回限りのゲーム体験

### 技術的特徴
- **Go言語**: シンプルで効率的な実装
